const LOG_LINE_REGEXP = /^\[(?<ts>[^\]]+)\] session=(?<session>\S+) direction=(?<direction>\S+) message=(?<message>".*")( .*)?$/;

De-escape rule (D-03 + CONTEXT specifics §ANSI):
- The `message` capture from the regex is a JSON-string literal (includes surrounding quotes).
- `JSON.parse(captured_message)` yields the REAL chunk text — real ANSI escape characters (`\u001b` char) and real `\r\n` sequences.
- Feed the `JSON.parse` result directly to `feedText`. Do NOT manually replace `\u001b` or `\r\n`; `JSON.parse` already handles it.

Output JSONL format (per D-11):
```
{"chunkIndex":0,"events":[]}
{"chunkIndex":1,"events":[{"kind":"room","room":{/* ParsedRoom */}}]}
{"chunkIndex":2,"events":[{"kind":"movement","direction":"north"}]}
```
One JSON object per line. Fields:
- `chunkIndex` — monotonic 0-based counter across ALL `direction=mud-in` lines processed (not per-session; the fixture is a single flat stream).
- `events` — array of `ParsedEvent` objects produced from that single `feedText(state, chunk)` call.

The parser state (`state`) is **shared** across all chunks (single `createParserState()` at script start, fed iteratively). This matches real server semantics — the parser accumulates stream state across chunks (lineBuffer, rawLineBuffer, pendingMobs, etc.).

Diff strategy (D-12):
- Byte-exact JSONL comparison. Read both files line-by-line, compare each pair.
- First differing line: report `line N: expected <X>, got <Y>`. Exit 1.
- If `after.jsonl` has more or fewer lines than `before.jsonl`: report the length mismatch. Exit 1.
- All lines match: exit 0.

The executor may implement diff by reading both files into arrays of strings and comparing with `===`. A library (jsondiffpatch / deep-diff) is NOT required for parser snapshot — lines are either byte-identical or they're a regression. This is the "strict byte-equality" regression definition from D-19.

<anti_patterns>
- DO NOT use `JSON.stringify(events, null, 2)` (pretty-print). JSONL requires one JSON object per line. Use `JSON.stringify(events)` (single line) only.
- DO NOT emit one line per EVENT. Per D-11, one line per CHUNK even if a chunk produces multiple events or zero events. Chunks with zero events still emit a JSONL line with `events: []` — the chunk boundary is the load-bearing signal.

    ### Core function — `generateSnapshot`

    ```typescript
    interface SnapshotEntry {
      chunkIndex: number;
      events: ReturnType<typeof feedText>;
    }

    async function generateSnapshot(fixturePath: string, outPath: string): Promise<{ chunksProcessed: number; malformedLines: number }> {
      mkdirSync(dirname(outPath), { recursive: true });

      const state = createParserState();
      const rl = createInterface({ input: createReadStream(fixturePath) });

      let lineNumber = 0;
      let chunkIndex = 0;
      let malformedLines = 0;
      const outLines: string[] = [];

      for await (const line of rl) {
        lineNumber++;
        if (line.length === 0) continue;
        const match = LOG_LINE_REGEXP.exec(line);
        if (!match || !match.groups) {
          malformedLines++;
          process.stderr.write(`parser-snapshot: WARNING skipping malformed line ${lineNumber}\n`);
          continue;
        }
        const direction = match.groups["direction"];
        if (direction !== TARGET_DIRECTION) continue;
        const messageLiteral = match.groups["message"];
        if (messageLiteral === undefined) continue;
        let chunkText: string;
        try {
          chunkText = JSON.parse(messageLiteral) as string;
        } catch (error: unknown) {
          malformedLines++;
          process.stderr.write(`parser-snapshot: WARNING skipping line ${lineNumber} — message not valid JSON\n`);
          continue;
        }
        if (typeof chunkText !== "string") {
          malformedLines++;
          process.stderr.write(`parser-snapshot: WARNING skipping line ${lineNumber} — decoded message not a string\n`);
          continue;
        }
        const events = feedText(state, chunkText);
        const entry: SnapshotEntry = { chunkIndex, events };
        outLines.push(JSON.stringify(entry));
        chunkIndex++;
      }

      writeFileSync(outPath, outLines.join("\n") + (outLines.length > 0 ? "\n" : ""));
      return { chunksProcessed: chunkIndex, malformedLines };
    }
    ```

    Critical:
    - `state` is a single `createParserState()` held across the entire loop. The parser is stateful per-stream; resetting would produce wrong output.
    - `for await (const line of rl)` with `createReadStream` handles the possibly-large fixture file without loading into memory.
    - JSONL is emitted as `lines.join("\n") + "\n"` (final newline mandatory; zero-entry output has no trailing newline — matches POSIX convention).
    - `JSON.stringify(entry)` emits one-line JSON — no indent.

    ### Diff function

    ```typescript
    function diffSnapshots(beforePath: string, afterPath: string): { equal: boolean; firstDiff: string | null } {
      if (!existsSync(beforePath)) {
        return {
          equal: false,
          firstDiff: `parser-snapshot: before file missing at ${beforePath} — run --write-initial first`,
        };
      }
      const beforeLines = readFileSync(beforePath, "utf8").split("\n");
      const afterLines = readFileSync(afterPath, "utf8").split("\n");
      if (beforeLines.length !== afterLines.length) {
        return {
          equal: false,
          firstDiff: `length mismatch: before=${beforeLines.length} lines, after=${afterLines.length} lines`,
        };
      }
      for (let i = 0; i < beforeLines.length; i++) {
        if (beforeLines[i] !== afterLines[i]) {
          return {
            equal: false,
            firstDiff: `line ${i + 1} differs:\n  before: ${beforeLines[i]}\n  after:  ${afterLines[i]}`,
          };
        }
      }
      return { equal: true, firstDiff: null };
    }
    ```

    async function main(): Promise<never> {
      try {
        const args = parseArgs(process.argv.slice(2));
        if (args.help) {
          process.stdout.write(USAGE + "\n");
          process.exit(0);
        }
        if (!existsSync(args.fixture)) {
