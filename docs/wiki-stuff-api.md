# Работа с базой вещей wiki.bylins.su

Документация для агентов, которым нужно искать предметы в Былины MUD.

---

## Базовый URL

```
https://wiki.bylins.su/stuff.php
```

---

## 1. Поиск предмета по названию

**URL:** `https://wiki.bylins.su/stuff.php?q=СЛОВО`

Параметр `q` — подстрока в названии предмета (регистронезависимо).

**Примеры:**
```
https://wiki.bylins.su/stuff.php?q=разбойничий
https://wiki.bylins.su/stuff.php?q=костяной+нож
https://wiki.bylins.su/stuff.php?q=сапог
```

**Результат:** HTML-список совпадений. Каждый предмет — ссылка вида:
```html
<a href="?id=2473">разбойничий ножик</a>
```

Из `href` извлекаем числовой ID предмета.

---

## 2. Получить карточку предмета по ID

**URL:** `https://wiki.bylins.su/stuff.php?id=НОМЕР`

**Примеры:**
```
https://wiki.bylins.su/stuff.php?id=2473   → разбойничий ножик
https://wiki.bylins.su/stuff.php?id=1140   → разбойничьи сапоги
https://wiki.bylins.su/stuff.php?id=1161   → пояс разбойника
```

**Что содержит карточка:**
- Название предмета
- Тип (броня, оружие, одежда и т.д.)
- Слот надевания (голова, ноги, ступни, пояс и т.д.)
- Материал
- АС / броня
- Урон (для оружия): формат `NdM (среднее)`
- Аффекты (ловкость, попадание, ускорение и т.д.)
- Ограничения класса (например: `!богатыри`, `!дружинники`)
- Место лоада (у большинства стартовых предметов — **«Неизвестно»**)

---

## 3. Фильтрация по аффекту

**URL:** `https://wiki.bylins.su/stuff.php?aff1=N`

Числовые коды аффектов видны в HTML формы на странице `stuff.php`.

Полезные аффекты для татя:
- `ловкость` — основной приоритет
- `попадание`
- `ускорение`
- `доблесть`

Можно комбинировать: `?aff1=N&aff2=M`

---

## 4. Фильтрация по слоту надевания

**URL:** `https://wiki.bylins.su/stuff.php?wear_at=СЛОТ`

Значения слотов (латиницей как в форме):
```
голову, ноги, ступни, кисти, пояс, плечи, запястья,
туловище, шею, правая.рука, левая.рука, обе.руки, колчан
```

**Пример — все предметы на ступни:**
```
https://wiki.bylins.su/stuff.php?wear_at=ступни
```

---

## 5. Комбинированный поиск

Параметры можно комбинировать:
```
https://wiki.bylins.su/stuff.php?wear_at=ступни&aff1=ловкость
```

---

## Известные ограничения

- **Поле «Место лоада»** у большинства стартовых предметов = **«Неизвестно»** — вики не заполнена
- Информацию о локациях лучше искать на форуме `bylins.su/vbb` или в описаниях зон на вики
- Форум `bylins.su/vbb` отдаёт страницы в кодировке **windows-1251** — при fetch нужен iconv или аналог
